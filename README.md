## QXDownloader

**QXDownloader**는 `yt-dlp`와 `Python`을 기반으로 제작된 강력하고 직관적인 유튜브 영상 및 재생목록 다운로더입니다.

---

### 한국어

**QXDownloader**는 유튜브의 개별 영상과 재생목록을 간편하게 다운로드할 수 있는 프로그램입니다. `yt-dlp`의 강력한 엔진을 활용하여 안정적인 다운로드를 지원하며, 사용자가 재생목록에서 원하는 영상만 선택하여 받을 수 있습니다. 작업 진행 상황을 퍼센트와 개수로 실시간 확인할 수 있으며, 비공개 영상은 자동으로 무시합니다. 내장된 업데이트 체커를 통해 항상 최신 상태를 유지할 수 있습니다.

### English

**QXDownloader** is a robust and user-friendly tool built on `Python` and `yt-dlp` for downloading individual YouTube videos and entire playlists. It leverages the powerful `yt-dlp` engine to ensure stable performance, allowing users to selectively download specific videos from any playlist. The tool provides real-time progress tracking (percentage and video count) and automatically skips private videos. With its built-in update checker, you can always stay up-to-date with the latest features and fixes.
